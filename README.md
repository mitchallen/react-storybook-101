# react-storybook-101

### Steps

```sh
# Assumes npm / npx installed
sudo npm uninstall -g create-react-app
npx clear-npx-cache
cd ~/projects/javascript
npx create-react-app react-storybook-101
cd react-storybook-101
ls -la
git log
npx storybook@latest init
npm run storybook
```
